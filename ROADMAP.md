# GridWatch Roadmap

## v1.0 — Open Source Launch

- [x] Security hardening (RLS, cron secrets, env example)
- [x] Documentation and contributor onboarding
- [x] CI/CD pipeline
- [ ] Public beta with community reporting

## v1.1 — Reliability

- [ ] Human-in-the-loop review queue for high-confidence AI node updates
- [ ] Materialized view / optimized read model for map nodes at scale
- [ ] Integration tests for report aggregation and AI node matching
- [ ] Structured logging and observability (Sentry/Datadog hooks)

## v1.2 — Notifications

- [ ] Push/email alerts for area outages
- [ ] Webhook API for third-party integrations
- [ ] DisCo-specific notification channels

## v2.0 — Platform

- [ ] Public REST/GraphQL API with API keys and rate limits
- [ ] Multi-region deployment guides
- [ ] Mobile app (React Native or PWA enhancements)
- [ ] Historical outage analytics and export
